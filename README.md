# Getting Started

### 1. `npm i`

### 2. `npm run dev`

# Available Scripts

### `npm run dev`

Runs the app in the development mode.

### `npm run build`

All OS builds in the `build` folder.

### `npm run build-l`

Only Linux OS build.

### `npm run build-m`

Only Mac OS build.

### `npm run build-w`

Only Windows OS build.
