# Getting Started

1. `npm i`
2. `npm run dev`

# Available Scripts

### `npm run dev`

Runs the app in the development mode.

### `npm run build`

All image builds in the `build` folder.

### `npm run build-l`

Linux image build in the `build` folder.

### `npm run build-m`

Mac image build in the `build` folder.

### `npm run build-w`

Windows image build in the `build` folder.
